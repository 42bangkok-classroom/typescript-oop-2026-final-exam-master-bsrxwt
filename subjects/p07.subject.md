# โจทย์ที่ 7: DELETE purchase (2 คะแนน)

ลบคำสั่งซื้อตาม `id`

**Path:** `DELETE /purchases/:id`

**ไฟล์ที่ต้องแก้ไข (จากข้อ 2):**

- `src/purchases/purchase.service.ts` (เพิ่ม method `remove(id: number)`)
- `src/purchases/purchase.controller.ts` (เพิ่ม endpoint `DELETE /purchases/:id`)

**เงื่อนไข:**

- อ่านข้อมูลจาก `data/purchases.json` เพื่อหาคำสั่งซื้อที่มี `id` ตรงกัน
- หากหาไม่พบ → ตอบ error response
- หากหาพบ → ลบออกจาก Array แล้วเขียนกลับไฟล์
- Response ต้องเป็นรูปแบบ `ApiResponse<T>`

### กรณีลบสำเร็จ:

**Request:**

```text
DELETE /purchases/1
```

**Response:**

```json
{
  "success": true,
  "data": null,
  "message": "Deleted purchase with id 1 successfully"
}
```

### กรณีไม่พบ id:

```json
{
  "success": false,
  "data": null,
  "message": "Purchase with id 1 not found"
}
```

**ข้อห้าม:**

1. ห้ามใช้ `process.exit()` มิเช่นนั้นจะตัดคะแนนเป็น 0 ทันที
2. ห้ามแก้ไข file `products.json` หรือ `purchases.json` หากใช้เพื่อทดสอบเสร็จแล้วให้แก้ไขเป็นข้อมูลเดิมหรืออย่า push ขึ้น git repository.
