# โจทย์ที่ 0: Hello (1 คะแนน)

สร้าง endpoint ตรวจว่าระบบทำงานได้

**Path:** `GET /`

**ไฟล์ที่ต้องแก้ไข:**

- `src/app.controller.ts` (class `AppController`)
- `src/app.service.ts` (class `AppService`)

**ไฟล์ที่ต้องแก้ไข:**

- `src/app.module.ts` (import และเพิ่ม `AppController`, `AppService`)

**เงื่อนไข:**

- Response ต้องเป็นรูปแบบ `ApiResponse<T>` ที่กำหนดไว้ใน `src/interfaces/response.interface.ts`
- data ต้องมี `service` เป็น `"purchase-api"` และ `version` เป็น `"1.0.0"`
- message ต้องเป็น `"Hello NestJS"`

**ตัวอย่าง Response:**

```json
{
  "success": true,
  "data": {
    "service": "purchase-api",
    "version": "1.0.0"
  },
  "message": "Hello NestJS"
}
```

**ข้อห้าม:**

1. ห้ามใช้ `process.exit()` มิเช่นนั้นจะตัดคะแนนเป็น 0 ทันที
2. ห้ามแก้ไข file `products.json` หรือ `purchases.json` หากใช้เพื่อทดสอบเสร็จแล้วให้แก้ไขเป็นข้อมูลเดิมหรืออย่า push ขึ้น git repository.
