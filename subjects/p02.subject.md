# โจทย์ที่ 2: GET all purchases (1 คะแนน)

สร้าง endpoint สำหรับดึงรายการคำสั่งซื้อทั้งหมด

**Path:** `GET /purchases`

**ไฟล์ที่ต้องสร้าง:**

- `src/purchases/purchase.module.ts` (class `PurchaseModule`)
- `src/purchases/purchase.controller.ts` (class `PurchaseController`)
- `src/purchases/purchase.service.ts` (class `PurchaseService`, method `findAll()`)
- `src/purchases/purchase.interface.ts` (interface `Purchase`, `PurchaseItem`)

**ไฟล์ที่ต้องแก้ไข:**

- `src/app.module.ts` (import และเพิ่ม `PurchaseModule`)

**เงื่อนไข:**

- อ่านข้อมูลจาก `data/purchases.json`
- Response ต้องเป็นรูปแบบ `ApiResponse<T>`
- message ต้องเป็น `"Fetched purchases successfully"`

**ตัวอย่าง Response:**

```json
{
  "success": true,
  "data": [
    {
      "id": 1,
      "customerName": "John Doe",
      "purchaseDate": "2024-01-15",
      "items": [
        { "productId": 1, "quantity": 2, "price": 25000 },
        { "productId": 2, "quantity": 5, "price": 500 }
      ],
      "totalPrice": 52500
    }
  ],
  "message": "Fetched purchases successfully"
}
```

**ข้อห้าม:**

1. ห้ามใช้ `process.exit()` มิเช่นนั้นจะตัดคะแนนเป็น 0 ทันที
2. ห้ามแก้ไข file `products.json` หรือ `purchases.json` หากใช้เพื่อทดสอบเสร็จแล้วให้แก้ไขเป็นข้อมูลเดิมหรืออย่า push ขึ้น git repository.
