# ข้อสอบแบบเลือกตอบ (10 คะแนน)

**จำนวน 20 ข้อ รวม 10 คะแนน (ข้อละ 0.5 คะแนน)**

ให้นักศึกษาเข้าไปอ่านคำถามในเอกสารที่แจกให้ แล้วตอบคำตอบลงใน `src/choices/answers.ts`

**วิธีการทำ:**

1. อ่านคำถามในเอกสารที่แจกให้
2. เลือกคำตอบที่ถูกต้อง (A, B, C, หรือ D)
3. บันทึกคำตอบลงใน `src/choices/answers.ts`

**รูปแบบการตอบ:**

```typescript
export const answers = {
  c01: 'A',
  c02: 'B',
  // ... ตอบทุกข้อ
};
```

**ข้อห้าม:**

1. ห้ามใช้ `process.exit()` มิเช่นนั้นจะตัดคะแนนเป็น 0 ทันที
