# ข้อสอบแบบเลือกตอบ

## คำนำ

ยินดีต้อนรับสู่ส่วนข้อสอบแบบเลือกตอบ! ส่วนนี้ทดสอบความเข้าใจของคุณเกี่ยวกับ TypeScript และแนวคิดการเขียนโปรแกรมเชิงวัตถุ (Object-Oriented Programming) ผ่านชุดคำถามแบบเลือกตอบ

คุณจะต้องเรียกดูโฟลเดอร์คำถามต่างๆ อ่านแต่ละคำถามอย่างระมัดระวัง เลือกคำตอบที่ดีที่สุด และบันทึกคำตอบของคุณในไฟล์ `answers.ts`

---

## วิธีการทำข้อสอบ

ทำตามขั้นตอนเหล่านี้เพื่อทำข้อสอบแบบเลือกตอบให้เสร็จสมบูรณ์:

1. **คำถาม**: คำถามจะอยู่ในเอกสารที่แจกให้
2. **เลือกคำตอบของคุณ**: เลือกคำตอบที่ดีที่สุดจากตัวเลือกที่ให้มา (A, B, C, หรือ D)
3. **บันทึกคำตอบของคุณ**: เพิ่มคำตอบของคุณลงในไฟล์ `answers.ts` ในโฟลเดอร์หลักของ `choices/`
4. **ตรวจสอบคำตอบของคุณ**: ตรวจสอบอีกครั้งว่าคำตอบทั้งหมดถูกบันทึกอย่างถูกต้องก่อนส่ง

---

## รูปแบบการตอบ

คำตอบทั้งหมดต้องถูกบันทึกในไฟล์ `answers.ts` โดยใช้รูปแบบต่อไปนี้:

```typescript
export const answers = {
  c01: 'A',
  c02: 'B',
  c03: 'C',
  // ... และทำเช่นนี้สำหรับคำถามทั้งหมด
};
```

### กฎการจัดรูปแบบ:

- **รหัสคำถาม**: ใช้ชื่อโฟลเดอร์คำถามเป็นคีย์ (เช่น `"c01"`, `"c02"`)
- **ค่าคำตอบ**: ใช้ตัวอักษรพิมพ์ใหญ่ตัวเดียว: `"A"`, `"B"`, `"C"`, หรือ `"D"`
- **รูปแบบสตริง**: ทั้งคีย์ (รหัสคำถาม) และค่า (คำตอบ) ต้องเป็นสตริง
- **ตอบทุกคำถาม**: ตรวจสอบให้แน่ใจว่าทุกคำถามมีคำตอบ (แม้ว่าจะเหลือเป็นสตริงว่าง `""` สำหรับคำถามที่ยังไม่ได้ตอบ)

---

## แนวทางการส่งคำตอบ

เมื่อคุณพร้อมส่งคำตอบแล้ว:

1. **บันทึกงานของคุณ**: ตรวจสอบให้แน่ใจว่าคำตอบทั้งหมดถูกบันทึกใน `answers.ts`
2. **ตรวจสอบรูปแบบ**: ตรวจสอบอีกครั้งว่าคำตอบของคุณตรงตามรูปแบบที่ถูกต้อง
3. **Commit การเปลี่ยนแปลง**: Commit ไฟล์ `answers.ts` ไปยังระบบควบคุมเวอร์ชัน
4. **Push ไปยัง repository**: Push commits ของคุณเพื่อส่งคำตอบสำหรับการให้คะแนน

> **หมายเหตุ**: ตรวจสอบให้แน่ใจว่าไฟล์ `answers.ts` ของคุณถูก commit และ push ก่อนหมดเวลาสอบ ไฟล์นี้เท่านั้นที่จะถูกใช้สำหรับการให้คะแนน

---

## ข้อควรระวัง

- **หนึ่งคำตอบต่อหนึ่งคำถาม**: แต่ละคำถามควรมีคำตอบที่เลือกเพียงหนึ่งเดียว
- **คำตอบที่ถูกต้องเท่านั้น**: ใช้เฉพาะตัวอักษร `A`, `B`, `C`, หรือ `D` (พิมพ์ใหญ่)
- **ไม่มีคำตอบบางส่วน**: แต่ละคำถามต้องมีตัวอักษรคำตอบที่สมบูรณ์
- **บันทึกบ่อยๆ**: บันทึกไฟล์ `answers.ts` ของคุณเป็นประจำเพื่อหลีกเลี่ยงการสูญเสียงาน
- **อ่านอย่างระมัดระวัง**: ตรวจสอบให้แน่ใจว่าคุณเข้าใจแต่ละคำถามก่อนเลือกคำตอบ
- **จัดการเวลา**: จัดสรรเวลาของคุณอย่างชาญฉลาดในทุกคำถาม

---

## ตัวอย่าง

นี่คือตัวอย่างที่สมบูรณ์ของการตอบคำถาม:

### ขั้นตอนที่ 1: อ่านคำถาม

ไปที่เอกสารที่แจกให้และอ่าน:

```markdown
# Choice 01 (C01)

Which statement best describes asynchronous code?
A. Code that always runs in parallel
B. Code that blocks execution until finished
C. Code that allows other work while waiting for an operation to complete
D. Code that only works with APIs
```

### ขั้นตอนที่ 2: เลือกคำตอบของคุณ

หลังจากอ่านและพิจารณาตัวเลือก คุณตัดสินใจว่าคำตอบที่ถูกต้องคือ **C**

### ขั้นตอนที่ 3: บันทึกใน answers.ts

อัปเดตไฟล์ `answers.ts`:

```typescript
export const answers = {
  c01: 'C', // คำตอบที่คุณเลือก
  c02: '', // ยังไม่ได้ตอบ
};
```

### ขั้นตอนที่ 4: ทำซ้ำสำหรับทุกคำถาม

ดำเนินกระบวนการนี้ต่อสำหรับทุกคำถามในข้อสอบ โดยอัปเดตไฟล์ `answers.ts` ไปเรื่อยๆ

---

## การขอความช่วยเหลือ

หากคุณพบปัญหาใดๆ:

- ทบทวนคำถามอย่างระมัดระวัง - บางครั้งคำตอบอยู่ในรายละเอียดของคำถาม
- ตรวจสอบว่ารูปแบบคำตอบของคุณตรงกับตัวอย่างด้านบน
- ตรวจสอบให้แน่ใจว่าไฟล์ `answers.ts` ของคุณมี syntax TypeScript ที่ถูกต้อง
- ตรวจสอบว่ารหัสคำถามทั้งหมดตรงกับชื่อโฟลเดอร์ทุกประการ

ขอให้โชคดีกับการสอบ! 🎯
